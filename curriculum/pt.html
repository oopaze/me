Projetos em que Contribuí

Connecta (Responsável)

Connecta é uma plataforma all-in-one projetada para gerenciar leads, integrando serviços como Facebook, Instagram, Telegram e WhatsApp. Fui o responsável por este projeto, que participou de vários desafios de startups. A plataforma consistia de um backend desenvolvido em TypeScript/Next.js, um serviço de chat criado com Python/Flask/MongoDB e um frontend em React. O chat e o frontend se comunicavam via REST API e WebSocket, enquanto o backend e o serviço de chat estavam conectados por filas RabbitMQ. Fui responsável por desenvolver o frontend e o serviço de chat, dando suporte a outro desenvolvedor que trabalhou no backend.

  • Tecnologias: TypeScript, Next.js, Python, Flask, MongoDB, React, RabbitMQ, WebSocket, REST API

Cloud Backups

Cloud Backups é uma plataforma que oferece serviços de backup de dados, garantindo a segurança dos dados mesmo se forem excluídos. Ela inclui dois serviços: o backend e os agentes. O backend foi desenvolvido em Django, e o frontend usou React com Next.js. O agente, escrito em Python, armazena dados em um banco de dados NoSQL. O backend e os agentes se comunicam através de filas RabbitMQ, e o Wasabi é utilizado para armazenar os dados de backup. Fui responsável por criar todo o backend e o agente de dispositivo, além de apoiar outros dois desenvolvedores no frontend como líder técnico.

  • Tecnologias: Django, React, Next.js, Python, Banco de Dados NoSQL, RabbitMQ, Wasabi
